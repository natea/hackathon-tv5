# Live Audience Demo Script

## Opening (2 minutes)

### Slide: "The Problem"
```
Most people avoid difficult family conversations because:
- They don't know how to start
- They fear escalation
- They predict rejection/abandonment
- They feel guilty for having needs
- They're trapped in old patterns

What if you could practice these conversations with AI 
that actually understands the emotional dynamics?
```

### Transition
"Today, we're going to practice 5 deeply difficult family conversations. 
The system will play the family member. You'll see how emotional analysis 
and real-time feedback can help you navigate these situations differently."

```

   Your hurt/anger/guilt = information, not weakness

   The system helps you recognize cycles you're in

   The goal isn't to "win" the conversation
   The goal is to communicate authentically & protect yourself

   Firmness + clarity + compassion = boundaries that stick
   
5. WHEN FAMILY SYSTEMS CAN'T CHANGE
   You can. You grieve what you didn't get.
   You choose what comes next.
```

## Technical Notes for Live Execution

### Required Displays:
- Main screen: Conversation transcript (left)
- Emotional analysis panel (right)
- Real-time metrics updating
- Alternative approach comparison

### Failsafes:
- Have 1-2 pre-recorded example conversations as backup
- Test API latency before event (emotional analysis needs to feel responsive)
- Have facilitator scripts ready if audience engagement drops
- Prepare simplified versions if system is slow

### Audience Engagement Checkpoints:
- After each scenario, ask for input
- Use polls/voting to keep people active
- Call on specific people to suggest responses
- Make it collaborative, not performative
